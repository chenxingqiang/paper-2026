# 方法论说明：合成控制法（Synthetic Control Method）

## 一、方法背景

### 1.1 发展历程

合成控制法（SCM）由Alberto Abadie及其合作者发展，主要文献包括：

| 年份 | 作者 | 论文 | 贡献 |
|------|------|------|------|
| 2003 | Abadie & Gardeazabal | AER | 首次提出SCM，研究巴斯克冲突的经济影响 |
| 2010 | Abadie, Diamond & Hainmueller | JASA | 系统阐述方法，研究加州控烟政策 |
| 2015 | Abadie, Diamond & Hainmueller | AJPS | 应用于比较政治学 |
| 2021 | Abadie | JEL | 方法综述，讨论应用条件和最新进展 |

### 1.2 适用场景

SCM特别适用于以下情形：
- **单一或少数处理单位**：传统DID需要多个处理单位，SCM可处理单一处理单位
- **宏观政策评估**：省级、国家级政策的效应评估
- **缺乏平行趋势**：处理组与控制组在政策前趋势不平行时

### 1.3 与DID的比较

| 特征 | 双重差分法（DID） | 合成控制法（SCM） |
|------|-------------------|-------------------|
| 处理单位数量 | 需要多个 | 可处理单一单位 |
| 平行趋势假设 | 要求严格平行 | 通过加权实现匹配 |
| 控制组选择 | 主观选择或全部使用 | 数据驱动的加权组合 |
| 透明度 | 控制组选择不透明 | 权重公开透明 |
| 适用层级 | 微观和宏观 | 主要宏观 |

## 二、方法原理

### 2.1 基本设定

假设观察到 $J+1$ 个单位在 $T$ 个时期的数据：
- 第1个单位（浙江省）在第 $T_0$ 期开始接受政策干预
- 其余 $J$ 个单位为潜在控制单位
- 时期 $t = 1, \ldots, T_0$ 为政策前时期
- 时期 $t = T_0, \ldots, T$ 为政策后时期

### 2.2 潜在结果框架

对于任意单位 $i$ 和时期 $t$：
- $Y_{it}^N$：没有政策干预时的潜在结果
- $Y_{it}^I$：接受政策干预时的潜在结果

政策效应定义为：
$$\tau_{it} = Y_{it}^I - Y_{it}^N$$

对于处理单位（$i=1$），我们观察到：
$$Y_{1t} = \begin{cases} Y_{1t}^N & \text{if } t < T_0 \\ Y_{1t}^I & \text{if } t \geq T_0 \end{cases}$$

**核心问题**：我们无法观察到 $Y_{1t}^N$（$t \geq T_0$），即浙江在没有成为示范区时的城乡收入差距。

### 2.3 合成控制的构建

**思想**：用控制单位的加权平均来近似处理单位的反事实结果。

设 $W = (w_2, \ldots, w_{J+1})'$ 为控制单位的权重向量，满足：
$$w_j \geq 0, \quad \sum_{j=2}^{J+1} w_j = 1$$

合成控制的结果变量：
$$\hat{Y}_{1t}^N = \sum_{j=2}^{J+1} w_j Y_{jt}$$

政策效应估计：
$$\hat{\tau}_{1t} = Y_{1t} - \hat{Y}_{1t}^N, \quad t \geq T_0$$

### 2.4 权重优化

**目标**：选择权重使合成控制组在政策前尽可能接近处理单位。

设：
- $X_1$：处理单位的预测变量向量（$k \times 1$）
- $X_0$：控制单位的预测变量矩阵（$k \times J$）
- $V$：预测变量权重矩阵（$k \times k$，正定对角阵）

**内层优化**（给定 $V$，优化 $W$）：
$$W^*(V) = \arg\min_W (X_1 - X_0 W)'V(X_1 - X_0 W)$$
$$\text{s.t. } w_j \geq 0, \sum_j w_j = 1$$

**外层优化**（优化 $V$）：
$$V^* = \arg\min_V \sum_{t=1}^{T_0-1} \left(Y_{1t} - \sum_j w_j^*(V) Y_{jt}\right)^2$$

## 三、统计推断

### 3.1 安慰剂检验（Placebo Test）

由于处理单位只有一个，传统的标准误估计不适用。Abadie等（2010）提出基于安慰剂检验的推断方法。

**空间安慰剂**：
1. 依次将每个控制单位假设为处理单位
2. 对每个"安慰剂处理单位"估计合成控制模型
3. 计算安慰剂效应
4. 比较真实效应与安慰剂效应分布

**p值计算**：
$$p = \frac{\#\{j: |\hat{\tau}_j| \geq |\hat{\tau}_1|\}}{J+1}$$

### 3.2 RMSPE比率

更严格的推断基于处理后与处理前RMSPE的比率：

$$\text{RMSPE比率} = \frac{\sqrt{\frac{1}{T-T_0+1}\sum_{t=T_0}^T \hat{\tau}_{1t}^2}}{\sqrt{\frac{1}{T_0-1}\sum_{t=1}^{T_0-1} \hat{\tau}_{1t}^2}}$$

该比率越大，表示政策后效应相对于政策前拟合误差越显著。

### 3.3 留一交叉验证

检验结果对特定控制单位的敏感性：
1. 依次排除权重较大的控制单位
2. 重新估计模型
3. 检验ATT估计值的稳定性

## 四、本研究的实现

### 4.1 变量选择

**结果变量**：城乡收入比（城镇居民人均可支配收入/农村居民人均可支配收入）

**预测变量**：
| 变量 | 说明 | 选择理由 |
|------|------|----------|
| 人均GDP | 经济发展水平 | 影响城乡收入的基础变量 |
| 城镇化率 | 城镇化水平 | 直接影响城乡人口分布和收入 |
| 第三产业占比 | 产业结构 | 影响就业结构和收入分配 |
| 人均财政支出 | 政府能力 | 影响公共服务和再分配 |
| 人均固定资产投资 | 投资水平 | 影响经济发展和就业 |

### 4.2 样本设定

| 项目 | 设定 |
|------|------|
| 处理单位 | 浙江省 |
| 控制单位 | 24个省份（排除直辖市和特殊地区） |
| 时间跨度 | 2010—2023年 |
| 政策时点 | 2021年 |
| 政策前时期 | 2010—2020年（11年） |
| 政策后时期 | 2021—2023年（3年） |

### 4.3 估计结果

**权重分配**：
- 江苏：52.1%
- 广东：42.0%
- 海南：5.9%

**模型拟合**：
- 处理前RMSE：0.0524
- 预测变量平均改善比例：95.1%

**政策效应**：
- 平均ATT：-0.106
- 相对效应：-4.64%
- p值：< 0.001

## 五、方法局限与注意事项

### 5.1 识别假设

SCM的因果识别依赖于以下假设：

1. **无预期效应**：政策实施前，处理单位的结果不受预期政策的影响
2. **无溢出效应**：政策对控制单位无影响（SUTVA）
3. **凸包假设**：处理单位的特征在控制单位特征的凸包内
4. **无未观测混杂**：影响结果的因素已被预测变量捕捉

### 5.2 应用注意

1. **控制单位选择**：应排除可能受到类似政策影响的单位
2. **预测变量选择**：应包含影响结果的重要变量
3. **时间跨度**：政策前时期应足够长以保证拟合质量
4. **政策后时期**：过短可能无法捕捉长期效应

### 5.3 本研究的局限

1. **政策后时期较短**：仅3年（2021—2023），长期效应需持续跟踪
2. **控制组可能受影响**：其他省份可能借鉴浙江经验，导致效应低估
3. **单一结果变量**：仅考虑城乡收入差距，未涵盖共同富裕其他维度

## 六、代码实现说明

### 6.1 核心算法

```python
# 伪代码
class SyntheticControl:
    def fit(self, data, outcome, predictors, treated_unit, treatment_time):
        # 1. 构建预测变量矩阵
        X0, X1 = self._build_predictor_matrix()
        
        # 2. 构建结果变量矩阵
        Z0, Z1 = self._build_outcome_matrix()
        
        # 3. 嵌套优化求解权重
        W, V = self._optimize_weights(X0, X1, Z0, Z1)
        
        # 4. 计算合成控制结果
        Y_synthetic = Z0 @ W
        
        # 5. 计算政策效应
        treatment_effect = Y_treated - Y_synthetic
        
        return self
```

### 6.2 优化算法

本研究采用嵌套优化：
- **内层**：给定V，求解W的二次规划问题（约束优化）
- **外层**：搜索最优V，最小化政策前预测误差

### 6.3 推断方法

```python
# 安慰剂检验
for placebo_unit in control_units:
    placebo_scm = SyntheticControl()
    placebo_scm.fit(data, treated_unit=placebo_unit)
    placebo_effects.append(placebo_scm.att)

# p值计算
p_value = mean(abs(placebo_effects) >= abs(true_effect))
```
